# desafio-fullstack-veritas
É um Mini Kanban com Go

---

## Para rodar o backend
<br>Para iniciar o Go, basta ir na pasta Backend e iniciar no cmd com o código a seguir: run go main.go
<br> <br>

---

## Para rodar o frontend
<br>Para iniciar o React, basta ir na pasta Frontend e iniciar no cmd com o código a seguir: npm start
<br> <br>

---

## Decições tomadas
<br> Eu busquei criar um design mais limpo e bonito, estando centralizado no meio da tela e fail de mexer e alterar
<br> <br>

---

## Limitações
<br>Como é simples, não vai conter histórico de tarefas apagadas, nem data da criação, modificação e conclusão
<br>Logo, o sistema parece mais algo simples, um editor de tarefas que tem como ficar movendo tarefas de lá para cá
<br>Outra limitação do programa é com relação a design, pois não permite alterar as cores e nem tamanho das telas e nem fazer outras alterações, sendo um programa estático.
